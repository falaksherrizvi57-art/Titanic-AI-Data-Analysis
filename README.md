AI Data Analytics: Titanic Survival Predictor

📌 Project Overview
This project analyzes the historic Titanic passenger dataset to uncover hidden survival patterns and trains a Machine Learning model to predict passenger survival based on age, gender, and ticket class.

📊 Key Insights Uncovered
* Class Disparity: 1st Class passengers had a 63% survival rate, compared to just 24% for 3rd Class.
* Gender Impact: 74% of female passengers survived, while only 19% of male passengers survived.

🤖 AI Model Performance
Built and trained a Decision Tree Classifier using Scikit-Learn that achieves an 80.45% accuracy in predicting passenger survival on unseen testing data.

🛠️ Tech Stack Used
* Python (Core Logic)
* Pandas & NumPy (Data Cleaning & Feature Engineering)
* Scikit-Learn (Machine Learning Model Training)
